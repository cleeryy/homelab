## Procedure

### Per-node

- [ ] Check current cluster health and backup status.
- [ ] Drain or pause non-essential workloads if needed.
- [ ] Apply updates.
- [ ] Reboot if required, validate platform afterwards.

### Per-service

- [ ] Snapshot or back up before updating.
- [ ] Update through the documented mechanism for the service.
- [ ] Smoke-test after update.
