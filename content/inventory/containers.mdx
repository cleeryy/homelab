## Conventions

- LXC for single-purpose system services. Docker compose stacks run inside a Docker host (LXC or VM), not directly on the Proxmox host.
- CT IDs follow the same per-role allocation idea as VMs; legacy IDs predating the convention are kept rather than renumbered.

## Notes

Containers marked **pending validation** must either be assigned a documented purpose or retired through [Operations > Decommissioning](/operations/decommissioning).
