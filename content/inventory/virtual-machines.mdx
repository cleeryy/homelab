## ID ranges currently in use

- `100–199` — ingress and small platform VMs.
- `200–299` — Docker host VMs for application workloads.
- `300–399` — Docker host VMs for the media stack.
- `400–499` — edge / network.
- `127` — historical, outside the per-role ranges (TrueNAS).
- `10000+` — lab desktops.

## OS notes

- The Docker host VMs run Ubuntu 24.04 (LTS releases observed: 24.04.2, 24.04.3).
- The Nextcloud host's OS line is reported in Beszel as `6.8.0-111-generic` (kernel string); the Ubuntu release line is not captured at the same place.
