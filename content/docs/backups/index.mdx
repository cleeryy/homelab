# Backups

## Stratégie globale

| Système | Méthode | Fréquence | Rétention | Destination |
|---------|---------|-----------|-----------|-------------|
| Proxmox VMs | PBS (Proxmox Backup Server) via TrueNAS | Quotidienne | 7 jours | TNAS-01 |
| Configuration Proxmox | vzdump manuel | Hebdomadaire | 4 semaines | TNAS-01 |
| Docker volumes | Script de backup | Quotidienne | 14 jours | TNAS-01 |
| Fichiers statiques | rsync | Quotidienne | 30 jours | TNAS-01 |
| Base de données | mysqldump / pg_dump | Quotidienne | 7 jours | TNAS-01 |

## Règles

- Toute donnée critique doit avoir au moins 2 copies sur des supports distincts
- Les backups sont isolés sur le réseau de stockage (iSCSI / NFS dédié)
- Test de restauration mensuel pour chaque catégorie de données
- Chiffrement des backups contenant des données sensibles
- Notification en cas d'échec d'un backup (alerte Proxmox ou script)
