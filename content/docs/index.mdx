# Documentation Homelab

Bienvenue sur la base de connaissance technique de l'infrastructure.

## Par où commencer

Consultez la sidebar pour naviguer entre les sections. Voici quelques points d'entrée :

- **[Architecture](/docs/overview)** — Vue d'ensemble et design de l'infrastructure
- **[Inventory](/docs/inventory)** — Inventaire complet du matériel et des VM
- **[Network](/docs/network)** — Plan réseau, VLANs et routage
- **[Services](/docs/services)** — Catalogue des services déployés

## Conventions

- Les décisions d'architecture sont documentées dans la section [ADR](/docs/decisions)
- Les procédures pas-à-pas sont dans les [Runbooks](/docs/runbooks)
- Chaque modification importante doit être accompagnée d'une mise à jour de la documentation
