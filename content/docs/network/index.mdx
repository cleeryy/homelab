# Network

## VLANs

| VLAN ID | Name | Subnet | Gateway | Description |
|---------|------|--------|---------|-------------|
| 1 | Native / LAN | 10.0.0.0/24 | 10.0.0.1 | Réseau local utilisateur (non isolé) |
| 10 | Management | 10.0.10.0/24 | 10.0.10.1 | Gestion des hyperviseurs et équipements |
| 20 | Storage | 10.0.20.0/24 | 10.0.20.1 | Traffic iSCSI / NFS / sync réplication |
| 30 | Services | 10.0.30.0/24 | 10.0.30.1 | Applications et services internes |
| 40 | IoT | 10.0.40.0/24 | 10.0.40.1 | Appareils connectés (isolé) |
| 50 | Guest | 10.0.50.0/24 | 10.0.50.1 | WiFi invités (accès Internet seulement) |
| 99 | DMZ | 10.0.99.0/24 | 10.0.99.1 | Services exposés sur Internet |

## DNS

- Serveur DNS primaire : OPNsense (10.0.0.1) — forwarder vers Cloudflare (1.1.1.1) / Quad9 (9.9.9.9)
- Serveur DNS secondaire : svc-dns-01 (10.0.30.2) — Unbound en mode forwarding
- Zones locales : résolution des noms des VM/CT via enregistrements statiques OPNsense + mDNS

## DHCP

| Scope | Subnet | Range | Lease |
|-------|--------|-------|-------|
| LAN | 10.0.0.0/24 | 10.0.0.100 — 10.0.0.200 | 24h |
| IoT | 10.0.40.0/24 | 10.0.40.100 — 10.0.40.200 | 48h |
| Guest | 10.0.50.0/24 | 10.0.50.100 — 10.0.50.200 | 2h |
| Services | 10.0.30.0/24 | Réservations statiques uniquement | — |
| Management | 10.0.10.0/24 | Réservations statiques uniquement | — |
| Storage | 10.0.20.0/24 | Réservations statiques uniquement | — |

Les équipements d'infrastructure (hyperviseurs, NAS, firewall) ont des adresses IP statiques configurées localement. Les VM/CT critiques ont des réservations DHCP statiques sur OPNsense.

## Règles de firewall (résumé)

| Source | Dest | Port | Action | Description |
|--------|------|------|--------|-------------|
| VLAN 10 (Mgmt) | Any | Any | ✅ Allow | Accès complet depuis le management |
| VLAN 30 (Services) | VLAN 20 (Storage) | 2049, 3260 | ✅ Allow | NFS / iSCSI depuis les services |
| VLAN 1 (LAN) | VLAN 30 (Services) | 443, 80 | ✅ Allow | Accès web aux services |
| VLAN 40 (IoT) | VLAN 1/30 | Any | ❌ Block | IoT isolé des réseaux internes |
| VLAN 40 (IoT) | WAN | 443, 80 | ✅ Allow | IoT autorisé à sortir sur Internet |
| VLAN 50 (Guest) | VLAN 1/10/30/40 | Any | ❌ Block | Invités isolés de tout réseau interne |
| VLAN 50 (Guest) | WAN | 443, 80 | ✅ Allow | Accès Internet uniquement |
| Any | VLAN 99 (DMZ) | 443 | ✅ Allow | Accès HTTPS aux services exposés |
