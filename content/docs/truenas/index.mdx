# TrueNAS

## Configuration générale

| Propriété | Valeur |
|-----------|--------|
| **Version** | TrueNAS Scale 24.04 (Dragonfish) |
| **Nom d'hôte** | TNAS-01 |

## Datasets

| Dataset | Compression | Dedup | Atime | Shares | Description |
|---------|-----------|-------|-------|--------|-------------|
| tank-01/vms | lz4 | off | off | iSCSI | Disques VM (extents) |
| tank-01/backups | lz4 | off | off | NFS | Sauvegardes Proxmox |
| tank-01/media | lz4 | off | off | SMB | Bibliothèque média |
| tank-01/data | lz4 | off | off | SMB | Partages utilisateur |
| tank-01/templates | lz4 | off | off | NFS | ISO et CT templates |
| tank-01/homes | lz4 | off | off | SMB | Home directories |

## Services

| Service | Statut | Interface | Description |
|---------|--------|-----------|-------------|
| **SMB** | 🟢 Actif | VLAN Storage (10.0.20.2) | Partage fichiers utilisateur et médias |
| **NFS** | 🟢 Actif | VLAN Storage (10.0.20.2) | Partage templates, sauvegardes Proxmox |
| **iSCSI** | 🟢 Actif | VLAN Storage (10.0.20.2) | Block storage pour VM Proxmox |
| **S3 (MinIO)** | ⏸️ Planifié | VLAN Services | Sauvegardes cloud locales |
