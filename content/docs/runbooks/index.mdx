# Runbooks

Les runbooks décrivent les procédures pas-à-pas pour les opérations courantes sur le homelab.

## Procédures disponibles

- [Reconstruction d'une VM](rebuild-vm) — Procédure de reconstruction complète d'une VM Proxmox

## Conventions d'écriture

- Chaque runbook suit un format standardisé : objectif, prérequis, durée estimée, procédure, rollback
- Les checklists `- [ ]` indiquent les étapes à cocher manuellement lors de l'exécution
- Les blocs de code contiennent les commandes exactes à exécuter
