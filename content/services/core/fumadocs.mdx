## Purpose

This Fumadocs site is the canonical documentation surface for the homelab. It is deployed from the `homelab-wiki` repository.

## Placement

- Hosted from the `homelab-wiki` repository.
- Deployed as a containerised Next.js application; placement is **pending** confirmation against the live ingress.

## Relation to Docmost

- Docmost runs as LXC `109` on `pve1` and is still present from the previous setup.
- Fumadocs replaces Docmost as the canonical documentation layer. Migrating remaining content out of Docmost is tracked in the [migration plan](/architecture/migration-plan).

## Build and run

- Stack: Next.js 16 + Fumadocs v16, pnpm.
- Build: `pnpm build`. Image is produced from the repository's `Dockerfile` (standalone Next.js output).
- Local dev: `pnpm dev` on port `3545`.

## Failure impact

- The wiki is stateless. Content lives in Git. A site outage does not lose data.
